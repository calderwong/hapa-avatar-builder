## Narration

AI agents can do useful work, then leave a transcript that loses sources, decisions, and ownership. Hapa turns the work into portable Card Documents with identity, relationships, and append-only learning.

Avatar Builder and its 3D Tarot Draw existed before Build Week. Using Codex in an operator-selected GPT-5.6 Sol session, I extended that foundation with this Stargate reference path.

Ordinary Cards become network coordinates. Create Cores and Lock Coordinates gives each Card its own Hypercore and a `card.created` receipt: ID, core, revision, and digest. That establishes custody—not mint, commerce, or canon.

I place Cards in a deliberate order and dial. Change the order and the deterministic namespace changes; camera and phone state do not. The portal projects that sealed Formation.

Save This Gate collapses the Formation into one portable Context Card. It stores safe scene commitments, never the private address or join capability. Restoring it rebuilds the exact Formation, disconnected.

After explicit human approval, the same Card moves through origin staging, Overwind acknowledgement, and a source-only `.hapaCatalog` projection. Catalog can discover and return the pinned revision without creating a second head.

A separate, expiring Gate Pass lets two isolated local peers verify the same Card and meet through Hyperswarm, Noise, and Protomux. This is a real encrypted local proof, not an internet-wide deployment claim.

Inside the Gate, Context Forge freezes selected evidence and a local model proposes a new, unminted Card. Three peer-blind Wisdom Cards examine that packet from different rules. Dissent remains visible, and value tradeoffs return to the human.

Codex helped trace the architecture, correct a wrong-shell rebuild, implement matching UI, API, and CLI surfaces, generate evidence, and diagnose a recursive-worker incident. GPT-5.6 was the build-time reasoning partner—not hidden runtime magic. It held the system model, tested claim boundaries, and turned mistakes into guards. The committed Stargate line is twenty-two commits across one hundred twenty-four files.

Hapa is not the company. It is the shared Card platform. Avatar Builder is one participant, showing how useful work can leave useful, attributable learning behind.
